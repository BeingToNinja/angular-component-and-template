模板语法

示例1: <input>

当用户在input中输入“Sally”时, DOM元素value属性将变成“Sally”.
但是, HTML的value属性仍然保持不变, 依旧为“Sarah”.
HTMl的value属性指定初始值, DOM的value属性是当前值. 在浏览器中打开调试终端(也可以点击下方按钮, 快速查看结果), 键入input.getAttribute('value')
和input.value可以查看对应的HTML属性和DOM属性.
当输入框的值发生变化后可以重复获取HTML和DOM的value属性查看有什么不同.

示例2: 禁用的按钮

默认情况下按钮是处于启用状态的, 当添加了disabled属性时, 该按钮会被禁用.
添加和删除disabled属性可以开启/禁用按钮, 但一切与该HTML属性的值无关, 这就是为什么无法通过``来启用按钮的原因.
要想控制按钮的状态, 必须设置DOM的disabled属性.

尽管可以从技术上设置[attr.disabled]属性绑定, 但是值的不同之处在于, DOM属性绑定需要为布尔值, 而其对应的HTML属性绑定取决于该值是否为null.
<input [disabled]="condition ? true : false">
<input [attr.disabled]="condition ? 'disabled' : null">
通常, 使用DOM属性绑定而不是HTML属性绑定, 因为前者更直观, 语法更短, 并且性能更高.
